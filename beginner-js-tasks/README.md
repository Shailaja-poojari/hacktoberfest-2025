#  Beginner JavaScript Tasks

##  Task 1: Reverse a String
- File: `task-1.js`
- Write a function to reverse a string.
- Example: `"hello"` → `"olleh"`

##  Task 2: Prime Number Check
- File: `task-2.js`
- Write a function to check if a number is prime.
- Example: `7` → `true`, `10` → `false`

##  Task 3: Fibonacci Sequence
- File: `task-3.js`
- Write a function to generate the first n numbers of the Fibonacci Sequence.
- Example: `n=6` → `[0,1,1,2,3,5]`

## Task 4: Count Vowels In a String
- File: `task-4.js`
- Write a function to count how many vowels are in a string.
- Example: `"javascript"` → `3`
